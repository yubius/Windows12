## 前言
最近看到 Windows 12 概念版(Power by PowerPoint)后深受启发，决定做一个Windows12网页版，就像 [Windows 11 网页版](https://win11.blueedge.me/)一样。

参考[前景规划](#前景规划)

精美的ui设计，流畅丰富的动画，各种高级的功能(相较于网页版)
> 移动端适配不太完善，开桌面版网站凑合着用吧 `>v-)o`
